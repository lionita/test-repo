## GitHub visibility checklist
If local changes are not visible on GitHub:
- Ensure commits exist locally: `git log --oneline`.
- Ensure a remote is configured: `git remote -v`.
- Push the branch: `git push -u <remote> <branch>`.
- Open or refresh the pull request for the pushed branch.
